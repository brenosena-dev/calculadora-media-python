# calculadora-media-python
# Calculadora de Média em Python

Projeto desenvolvido para praticar lógica de programação utilizando Python.

## 📌 Funcionalidades
- Recebe notas digitadas pelo usuário
- Calcula a média
- Informa se o aluno foi aprovado ou reprovado

## 🛠 Tecnologias utilizadas
- Python

## 🎯 Objetivo
Projeto criado com foco em aprendizado e desenvolvimento da base em programação para estágio na área de Tecnologia.

## ▶️ Como executar
1. Baixe o arquivo `calculadora_media.py`
2. Execute no Python
3. Digite as notas quando solicitado
